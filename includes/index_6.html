Comments

Comments

  • Code quickly becomes confusing.
  • Everyone codes differently, which leads to the fact that the program of another can be difficult to understand.
  • Often it is not clear for variables what they stand for.
    • public class Comments {
    public static void main(String[] args) {
        int a = 20; 
    }
}
  • What exactly is stored in "a"?

Comments

  • These problems can be solved with comments!
    • public class Comments {
    public static void main(String[] args) {
        // a is the age of the user
        int a = 20; 
    }
}
  • They make the source code more understandable.
  • Comments do not change the flow of the program. They are only there for the programmer to understand.

Comments - Single-line

  • Single-line comments start with "//".
  • Everything that comes after "//" is not evaluated by the compiler and does not change the program flow.
  • Most of the time, single-line comments explain the next line of code.
    • public class SingleLineComment {
    public static void main(String[] args) { 
        // This is a single-line comment
        // The next line of code will output some text 
        System.out.println("Hello World"); 
        // This is also a comment
    }
}

Comments - Multi-Line

  • Comments over multiple lines start with "/*" and end with "*/".
  • Eclipse writes a "*" at the beginning of each line in between.
  • This is used for longer comments.
    • public class MultiLineComment {
    /*
    * This is a multi-line comment.
    * The code outputs "Hello world!" on the console.
    */
    public static void main(String[] args) { 
        System.out.println("Hello World"); 
    }
}

Comments - Multi-Line

public class Calculator { 
    /* 
    * The code calculates the number of hours per week. 
    * The result is the output.
    */ 
    public static void main(String[] args) { 
        // Number of hours per day
        int hours = 24; 
        // Number of days per week 
        int days = 7; 
        // Calculate hours per week
        int result = hours * days; 
        // Output: Number of hours per week
        System.out.println("Number of hours per week: " + result); 
    } 
}

Exercise 14 - "Comments III"

  • We want to write a program that converts the temperature from Celsius to Fahrenheit.
  • Create two variables (celsius and fahrenheit) of type double and initialize celsius with any value.
  • Output the temperature in Fahrenheit on the console.
  • Add appropriate comments to the code!
  • The formula for the conversion is:

    fahrenheit = celsius * 1.8 + 32

Exercise 14 - "Comments III"

/*
*  Name:  Max Mustermann
*  Date: 01.05.2020
*/
class Exercise14 {
    public static void main(String[] args) {
        //Variable for the temperature in Celsius 
        double celsius = 23; 
        //Variable for the temperature in Fahrenheit
        double fahrenheit = celsius * 1.8 + 32;
        //Output the result to the console 
        System.out.println("Temperature in Fahrenheit: " + fahrenheit);
    }
}
