Variables - Output

  • Variables can be printed to the console using the System.out.println() method.
  • The name of the variable must be transmitted as a parameter (between the brackets).
    • String name = "Max Mustermann";
System.out.println(name);
  • With "+" several words and variables can be appended to each other.
    • String name = "Max Mustermann";
int age = 23;
System.out.println(name + " is " + age + " years old.");

Variables - Output

  • Be careful when adding numbers without text!
    • int number1 = 4;
int number2 = 3;
System.out.println(number1 + number2);
  • zahl1+zahl2=4+3=7
  • So the numbers are added and not output one after the other.

Variables - Output

  • To get around this problem we add an empty text.
    • int number1 = 4;
int number2 = 3;
System.out.println("" + number1 + number2);
  • ""+zahl1+zahl2=43
  • The numbers are printed one after the other.
