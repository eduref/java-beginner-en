While loop

  • This is what the While loop is for.
  • A block of code that is executed as long as a condition is met.

While loop Syntax

  • In the code it looks like this:

    • while(LogicalExpression) {
    //Code block
}

  • This is how it works:
    1. The logical expression is checked.
    2. If the expression is true, the code block is executed and the program jumps back to the 1st step.
    3. If the expression is false, the while-loop is terminated.
