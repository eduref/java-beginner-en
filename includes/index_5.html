Exercise 10 - "Calculation II"

  • Write a program that calculates the BMI.
  • Declare an integer variable for body weight and a double variable for body height and store your values in it.
  • Calculate your BMI with the two values and output it to the console.

  • Use the following formula for this:

    BMI = body height in meters / (body weight in kg)²

  • N ote:

    number² = number * number

Exercise 10 - "Calculation II"
(Solution)

public class Exercise10{ 
    public static void main(String[] args){
        double bodyWeight, bodyHeight, bmi;
        bodyWeight = 85;
        bodyHeight = 1.80;
        bmi =  bodyHeight / (bodyWeight * bodyWeight);
        System.out.println(bmi);
    } 
}

Exercise 11 - "Variable swap"

  • While checking the program from exercise 10, you noticed that you swapped height and body weight in the formula.
  • Actually, the formula should look like this:

    BMI = body weight in kg / (body height in m)²

  • Do not correct the formula, but perform a variable swap to correct this!

Exercise 11 - "Variable swap"
(Solution)

public class Exercise11{ 
    public static void main(String[] args){        
        double bodyHeight, bodyWeight, bmi, temporaryVariable;
        bodyWeight = 85;
        bodyHeight = 1.80;
        
        temporaryVariable = bodyWeight;
        bodyWeight = bodyHeight;
        bodyHeight = temporaryVariable;

        bmi =  bodyHeight / (bodyWeight * bodyWeight);
        System.out.println(bmi);
    } 
}
