Scanner

Scanner

  • Until now, we had to assign all values to the variables when programming.
  • Now we want to read values (strings and numbers) from the keyboard.
  • For this we use a scanner.

Scanner - In code

  • First we need to import the scanner.
  • For this, the following line must be above the class:
    • import java.util.Scanner;

Scanner - In code

  • Then we need to create a scanner.
  • Only one scanner is needed per program.
  • For this we need the following line:
    • Scanner myScanner = new Scanner(System.in);

freely selectable

Scanner - Usage

  • Numbers and texts can be read in via the name of our scanner (here: myScanner).
  • Command for reading strings (text):
    • myScanner.next();
  • Command to read in integer (whole number):
    • myScanner.nextInt();
  • Read-in values should be stored in variables. Otherwise the values will be lost.

Scanner - Example

import java.util.Scanner;

public class ScannerExample {
    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in); 
        System.out.println("Please enter your name!");
    
        //Reading a name from the keyboard 
        String name = scanner.next();

        System.out.println("Name: " + name);
    }
}

15 - "Scanner I"

  • Write a program that creates a scanner.
  • Create two matching variables name and age.
  • Use the scanner to read the user's name and age.
  • Print name and age on the console.

Exercise 15 - "Scanner I"
(Solution)

import java.util.Scanner;
public class Exercise15 {
    public static void main(String[] args) { 
        Scanner scanner = new Scanner(System.in);
        String name; 
        int age;
        name = scanner.nextLine(); 
        age = scanner.nextInt(); 
        System.out.println("Name: " + name); 
        System.out.println("Age: " + age);
    }
}

Exercise 16 - "Scanner II"

  • Modify the program from exercise 14 (Celsius to Fahrenheit converter) so that the temperature is read in from the keyboard.
  • Ask the user to enter the temperature in Celsius.

    }
}

Exercise 16 - "Scanner II"
(Solution)

import java.util.Scanner;
class Exercise16 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Please enter temperature in Celsius!");
        //Variable for temperature in Celsius 
        double celsius = scanner.nextInt(); 
        //Variable for temperature in Fahrenheit
        double fahrenheit = celsius * 1.8 + 32;
        //Output of the result on the console 
        System.out.println("Temperature in Fahrenheit: " + fahrenheit);
    }
}
