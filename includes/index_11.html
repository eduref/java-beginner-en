Arrays - Current Status

  • We use elementary data types in which we can store a value.
  • Now we want to store a lot of data.
  • For this we would have to create a separate variable for each date.
    • int number1 = 10;
int number2 = 15;
int number3 = 20;
int number4 = 25;
int number5 = 30;
  • This is way too much effort!

Arrays

  • This is much easier with arrays.
  • Basically, several variables are stored within one array.

Theory

In code



int[] array = new int[5];
array[0] = 10;
array[1] = 15;
array[2] = 20;
array[3] = 25;
array[4] = 30;

  • The initialization is still too much of a writing effort!

Arrays

  • We can still shorten the initialization.
  • If we already know the values, we can directly initialize the array.
    • int[] array = {10, 15, 20, 25, 30};
  • The size is determined by the number of numbers in the curly brackets.

Arrays - Multidimensional

  • Sometimes you need more than one dimension (an array of arrays).
  • With further [] the dimension of arrays can be increased.

Theory

In code



int[][] array = new int[3][2];
array[0][0] = 10;
array[0][1] = 15;
array[1][0] = 20;
array[1][1] = 25;
array[2][0] = 30;
array[2][1] = 35;
10 15
20 25
30 35

Arrays - Multidimensional

  • Multidimensional arrays can also be initialized directly.
  • Then it is also possible to vary the sizes of the arrays in the lower dimensions.

Theory

In code



int[][] array;
array = {{10,15},{20,25,30},{35}};
10 15
20 25 30
35

Exercise 32 - "Arrays IV"

  • Ask for an integer value and read it in.
  • Create a one-dimensional array of the length of the read in value.
  • Initialize the array with the index of the position.
    • feld[i] = i;
  • Output the field on the console.
  • Note: the size of an array can be determined using ".length".
    • feld.length

Exercise 32 - "Arrays IV"
(Solution)

import java.util.Scanner;
class Exercise32 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Please enter the size of the array!");
        int size = scanner.nextInt();
        int[] array = new int[size];
        for(int i = 0; i < array.length; i++) {
            array[i] = i;
        }
        for(int i = 0; i < array.length; i++) {
            System.out.println(array[i]);
        }
    }
}

Exercise 33 - "Arrays V"

  • Code a name list.
  • First ask how many names should be stored.
  • Then create a string array with the respective size.
  • Ask for the names to initialize the array.
  • In an endless loop, ask for an index that should be used to output the value stored in the array at this index.

  • Note: an endless loop is a while-loop with true as its condition.

Exercise 33 - "Arrays V"
(Solution)

import java.util.Scanner;
class Exercise33 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Please enter the number of names!");
        int size = scanner.nextInt();
        String[] name = new String[size];
        for(int i = 0; i < name.length; i++) {
            System.out.println("Name :");
            name[i] = scanner.next();
        }
        while(true) {
            System.out.println("Which entry would you like to see?");
            int entry = scanner.nextInt();
            System.out.println("Name: " + name[entry]);
        }
    }
}
