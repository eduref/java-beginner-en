For-Loop Syntax

  • This is what the code looks like:
    • for(initialization; boolean valued expression; update) {
    //Code-Block
}

  • This is how it works:
    1. The initialization is executed.
    2. The boolean-valued expression is checked.
    3. If the expression is true, the code block is executed, the update is executed and the program jumps back to the 2nd step.
    4. If the expression is false, the while-loop is terminated.

Exercise 27 - "For-Loop II"

  • Add up the numbers 1 to 10
  • Output the result to the console.
  • Note: A variable "sum" is required.

  • Reminder:

    • class Exercise26 {
    public static void main(String[] args) {
        for (int number = 1; number < 10; number += 1) {
            System.out.println(number);
        }
    }
}

Exercise 27 - "For-Loop II"
(Solution)

class Exercise27 {
    public static void main(String[] args) {
        int sum = 0;
        for (int number = 1; number <= 10; number += 1) {
            sum += number;
        }
        System.out.println("The result is: " + sum);
    }
}
