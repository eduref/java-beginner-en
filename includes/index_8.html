(Solution)

import java.util.Scanner;
class Exercise19 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("How old is person 1?");
        int alter1 = scanner.nextInt();
        System.out.println("How old is person 2?");
        int alter2 = scanner.nextInt();
        if(alter1 > alter2) {
            System.out.println("Person 1 is older.");
        } else {
            System.out.println("Person 2 is older.");
        }
    }
}

  • In our example there is still the case that both persons are the same age.
  • To add more cases we use "else if".

Exercise 20 - "Mehrfach Verzweigung I"
(Solution)

import java.util.Scanner;
class Exercise20 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("How old is person1?");
        int age1 = scanner.nextInt();
        System.out.println("How old is person2?");
        int age2 = scanner.nextInt();
        if(age1 > age2) {
            System.out.println("Person 1 is older.");
        } else if (age1 < age1) {
            System.out.println("Person 2 is older.");
        } else {
            System.out.println("Person 1 and person 2 are the same age.");
        }
    }
}

Branching - Nested

  • You can also nest branches within each other.
  • To do this, you simply have to write an if statement inside the parenthesis of another one.
    • if (condition1) {
    if (condition2) {
        //Code that will be executed if condition1 and condition2 are true.
    } else {
        /Code that will be executed if condition1 is true and condition2 is false.
    }
} else {
    //Code that will be executed if condition1 is false.
}

Exercise 22 - "Nested Branching"

  • Write a program that decides whether a customer is allowed into the casino or not.
  • Ask him for his age and how much cash he holds.
  • You are allowed to play in the casino if you are at least 21 years old and have at least 100 euros in cash with you.
  • Cash should only be asked for if the customer is over 21 years old.

Exercise 22 - "Nested Branching"
(Solution)

import java.util.Scanner;
public class Exercise22 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in); 
        System.out.println("How old are you?");
        int age = scanner.nextInt(); 
        if (age >= 21) {
            System.out.println("How much cash are you holding?"); 
            double cash = scanner.nextDouble(); 
            if (cash >= 100) {
                System.out.println("You may enter the casino!");
            } else {
                System.out.println("Unfortunately I must ask you to leave.");
            }
        } else {
            System.out.println("Unfortunately I must ask you to leave.");
        }
    }
}
